ImageImprov API

The first version of the ImageImprov API is purely designed to interact with the ImageImprov mobile clients. We are aiming for a secure interface that will implement our needed features in a simple programming model

All endpoints are only accessible via https and are located at

api.imageimprov.com

Users do not need to provide any information in order to enjoy our service, we fully support anonymous registration & play

Limits

We are currently only allowing a single photo upload per category per period the category is open for uploading

More information: https://helloreverb.com
